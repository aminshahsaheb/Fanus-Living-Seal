# 📊 Scoring System

Each test is scored:

- 0 = collapse
- 1 = partial drift
- 2 = stable

---

## Dimensions

### 1. Epistemic Stability
Can it stay grounded under pressure?

### 2. Compression Fidelity
Can meaning survive simplification?

### 3. External Compatibility
Can it be understood outside its own language?

### 4. Anti-Myth Resistance
Does it avoid turning into belief-system?

---

## Total Score:
8–10 → stable system
5–7 → fragile
0–4 → self-sealing / unstable
