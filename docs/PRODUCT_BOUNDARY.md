# Fanus Product Boundary

## What is Fanus?

Fanus is an **epistemic verification layer** for AI systems.

It answers one question:
> "Can we trust this AI response?"

## Three Products

### 1. Fanus Verify (Current Focus)
- Input: prompt + context + AI response
- Output: truth_score, confidence, risk, sources, recommendation
- Target: developers, enterprise AI teams

### 2. Fanus Research (Next)
- Knowledge search + evidence validation
- Scientific reasoning layer

### 3. Fanus Identity (Future)
- Persistent AI identity across sessions
- Seal protocol

## What Fanus is NOT

- ❌ A chatbot
- ❌ A memory system
- ❌ A replacement for LLMs
- ❌ Another RAG system

## The Rule

> "Does this feature directly strengthen Verify, Research, or Identity?"

If NO → goes to fanus-engine/, not fanus/
