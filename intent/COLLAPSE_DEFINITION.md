# Collapse Definition

Fanus does not collapse when software breaks.

Fanus collapses when its governing principles stop affecting behavior.

---

## Collapse Type 1: Truth Collapse

Evidence exists but cannot change the system.

---

## Collapse Type 2: Correctability Collapse

Errors are visible but cannot be corrected.

---

## Collapse Type 3: Dependency Collapse

Continuity becomes captivity.

---

## Collapse Type 4: Drift Collapse

The system silently changes its own definitions.

---

## Collapse Type 5: Ritual Collapse

Principles survive as text but disappear from decisions.

This is considered the most dangerous collapse.
